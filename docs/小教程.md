## 1. 技能页面 (`src/data/skills.ts`)

展示你的技术栈和熟练程度。

### 数据结构

```typescript
{
  id: "唯一标识",           // 字母数字，不要有空格
  name: "技能名称",          // 显示的名字，如 "Go"
  description: "一段描述",    // 简短介绍
  icon: "图标名",           // 使用 Iconify 图标，如 "logos:go"
  category: "分类",         // "frontend" | "backend" | "database" | "tools" | "other"
  level: "等级",            // "beginner" | "intermediate" | "advanced" | "expert"
  experience: {
    years: 数值,            // 经验年数
    months: 数值            // 经验月数
  },
  projects: ["项目ID"],     // 可选，关联到 projects.ts 中的项目
  color: "#十六进制"        // 可选，技能卡片的主题色
}
```

### 示例

```typescript
{
  id: "go",
  name: "Go",
  description: "企业级后端开发，熟悉 Gin 框架与并发编程",
  icon: "logos:go",
  category: "backend",
  level: "advanced",
  experience: { years: 2, months: 6 },
  color: "#00ADD8"
}
```

### 可用分类

| 分类值 | 说明 |
|--------|------|
| `frontend` | 前端（HTML/CSS/JS/React 等） |
| `backend` | 后端（Go/Java/Python 等） |
| `database` | 数据库（MySQL/Redis/MongoDB 等） |
| `tools` | 工具（Docker/Git/Linux 等） |
| `other` | 其他 |

### 可用等级

| 等级值 | 说明 |
|--------|------|
| `beginner` | 入门 |
| `intermediate` | 中级 |
| `advanced` | 高级 |
| `expert` | 熟练/专家 |

### 找图标

在 [Iconify](https://icon-sets.iconify.design/) 搜索图标，复制格式如 `logos:go`、`vscode-icons:file-type-go`。

## 2. 日记页面 (`src/data/diary.ts`)

展示你的生活碎碎念。

### 数据结构

```typescript
{
  id: 编号,                 // 数字，按顺序递增
  content: "内容",           // 日记正文（支持纯文本）
  date: "日期",             // ISO 格式，如 "2025-01-15T10:30:00Z"
  images: ["图片路径"],      // 可选，图片数组
  location: "位置",          // 可选，地点
  mood: "心情",             // 可选，如 "happy"、"sad"
  tags: ["标签"]            // 可选，标签数组
}
```

### 示例

```typescript
{
  id: 1,
  content: "今天终于把博客搭好了，发一篇纪念一下！",
  date: "2025-06-01T12:00:00Z",
  location: "家里",
  mood: "开心",
  tags: ["博客", "日常"]
}
```

### 图片存放

图片放在 `public/images/diary/` 目录下，路径写为 `/images/diary/xxx.webp`。

## 3. 番剧页面 (`src/data/anime.ts`)

展示你的追番列表。

### 数据结构

```typescript
{
  title: "番剧名称",
  status: "状态",          // "watching" | "completed" | "planned"
  rating: 评分,            // 1~10 分
  cover: "封面路径",        // 如 "/assets/anime/xxx.webp"
  description: "简短描述",
  episodes: "集数",         // 如 "12 episodes"
  year: "年份",            // 如 "2025"
  genre: ["类型"],          // 如 ["Action", "Comedy"]
  studio: "制作公司",
  link: "链接",            // Bilibili 或其他网站链接
  progress: 看到第几集,     // 数字
  totalEpisodes: 总集数,    // 数字
  startDate: "开始日期",    // 如 "2025-01-01"
  endDate: "结束日期"
}
```

### 示例

```typescript
{
  title: "葬送的芙莉莲",
  status: "completed",
  rating: 9.5,
  cover: "/assets/anime/frieren.webp",
  description: "勇者一行击败魔王后的故事",
  episodes: "28 episodes",
  year: "2023",
  genre: ["Fantasy", "Adventure"],
  studio: "MADHOUSE",
  link: "https://www.bilibili.com/bangumi/media/md...",
  progress: 28,
  totalEpisodes: 28,
  startDate: "2023-10-01",
  endDate: "2024-03-01"
}
```

### 封面图片

放在 `public/assets/anime/` 目录下，路径写为 `/assets/anime/xxx.webp`。

如果使用 Bangumi 或 Bilibili 模式（在 `src/config.ts` 设置 `anime.mode`），数据会自动从 API 获取，但当前配置的是 `local` 模式，使用本地文件。

{

## 💡 通用技巧

### 1. 让修改生效

编辑完数据文件后，保存即可。开发服务器（`pnpm dev`）会自动热更新，不需要重启。

### 2. 图片路径规则

- 以 `/` 开头 → 图片放在 `public/` 目录下（如 `/assets/anime/xxx.webp` → `public/assets/anime/xxx.webp`）
- 不以 `/` 开头 → 图片放在 `src/` 目录下（如 `assets/images/avatar.webp` → `src/assets/images/avatar.webp`）

### 3. 图标资源

- **Iconify 图标**：在 [icon-sets.iconify.design](https://icon-sets.iconify.design/) 搜索，格式如 `logos:go`、`mdi:github`、`fa7-brands:github`
- **Material Symbols 图标**：格式如 `material-symbols:home`、`material-symbols:school`

### 4. 如果页面不想显示

在 `src/config.ts` 中关闭对应的 featurePages 开关即可：

```typescript
featurePages: {
  anime: false,      // 关闭番剧页面
  diary: false,      // 关闭日记页面
  friends: false,    // 关闭友链页面
  projects: false,   // 关闭项目页面
  skills: false,     // 关闭技能页面
  timeline: false,   // 关闭时间线页面
  albums: false,     // 关闭相册页面
  devices: false,    // 关闭设备页面
}
```

关闭后最好也在 `navBarConfig.links` 中移除对应的导航栏链接。

## 🎯 示例：完整替换技能数据

如果你想完全替换技能数据，操作如下：

**第 1 步**：打开 `src/data/skills.ts`

**第 2 步**：找到 `skillsData` 数组（大约第 20 行）

**第 3 步**：将数组内容全部替换为你自己的技能，例如：

```typescript
export const skillsData: Skill[] = [
  {
    id: "go",
    name: "Go",
    description: "后端开发，熟悉 Gin/Grpc",
    icon: "logos:go",
    category: "backend",
    level: "advanced",
    experience: { years: 2, months: 0 },
    color: "#00ADD8"
  },
  {
    id: "vue",
    name: "Vue.js",
    description: "前端开发，熟悉组合式 API",
    icon: "logos:vue",
    category: "frontend",
    level: "intermediate",
    experience: { years: 1, months: 6 },
    color: "#4FC08D"
  },
  // ... 继续添加更多技能
];
```

**第 4 步**：保存文件，浏览器会自动刷新。

其他页面（日记、番剧、友链等）的替换方法完全一致——找到对应的数据数组，替换成你的内容即可。
